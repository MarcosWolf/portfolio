# Como executar o projeto

```bash
# clonar repositório
git clone https://github.com/MarcosWolf/portfolio

# entrar na pasta do projeto
cd portfolio

# executar o comando
`npm start`

Acesse [http://localhost:3000](http://localhost:3000) para visualizar no seu navegador.
```
